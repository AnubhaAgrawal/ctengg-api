ctengg API is an unofficial REST API for the official ctengg website. This site provides the much needed constant end points for client developers who want to create their own web or mobile applications featuring the data from the said website and can't due to constantly changing page layout and url structure. We decided that instead of having to change the hardcoded parsing logic inside the app, why not build an online parsing web application which provides a robust, structured way to provide an interface to the result and attendance data to the app developers. So we did.

Why use it?
  • Better Performance
  • Well designed and static end points
  • Provides data in easily parselable JSON format
  • Easy implementation in applications
  • Extremely small response size
  • Continuously changing parser providing consistency

The response data is in JSON format and can be easily parsed and converted into desired object in plenty of languages like Java, Python, C (No, lol)!
If creating android applications, we recommend JSON2POJO to convert the provided JSON data into a Plain Old Java Object (POJO).
(Though, we encourage you to look out for any stray field that may slip through the response, our main motto is to keep the response very strict and constant.)
And to parse the response into that class, we recommend retrofit.
launch JSON2POJO launch Retrofit

Our response time is multitude's less than the front-end site, because we strip all the resources like images and other boilerplate code to ensure just necessary data gets through to you.
